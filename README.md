# Interactive Gantt Chart Creator in Shiny
## Overview
This Shiny application is an innovative tool for creating interactive Gantt charts. It's designed for ease of use in project planning, allowing users to define tasks, set durations, and visualize their project's timeline.

## Features
* User-Friendly Chart Creation: Initiate your project by naming it and then add tasks as required.
* Interactive Task Management: Tasks can be easily added or removed based on project needs.
* Simple Date Selection: Select start and end dates for each task through an interactive interface.
* Exportable Data: The task list can be downloaded as a CSV file.
* Chart as Image: The Gantt chart can be saved as an image directly from the interface.

## Hosted on GitHub with Shinylive
This app is uniquely hosted on a GitHub page, leveraging the shinylive package. This means you can interact with the app directly on GitHub, without the need for it to be hosted on a separate R server.
